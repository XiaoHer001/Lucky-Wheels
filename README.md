# 🎯 Unity 年会抽奖工具使用说明

## 1. 功能概述
本程序为公司内部抽奖工具，提供以下功能：

- 🧑‍💼 **员工及奖项管理**  
  支持从 Excel 文件导入员工名单及奖项信息。
- 🎯 **内定与随机抽奖**  
  系统可区分内定获奖人员和随机抽奖人员，保证内定奖项优先。
- 🎡 **转盘显示与动画**  
  抽奖通过动态转盘显示奖项，提供直观视觉反馈。
- 📄 **中奖记录自动保存**  
  每次抽奖结果自动记录到 Excel 文件，保留完整数据和时间戳。
- 🔊 **音效及背景音乐**  
  提供抽奖音效及背景音乐，可自由切换。
- ⌨️ **快捷键支持**  
  - **F5**：重新加载员工和奖项数据  
  - **Esc**：停止转盘  
  - **空格键**：切换背景音乐

---

## 2. 使用方法

### 2.1 准备工作
1. 将员工名单及奖项 Excel 文件放置在程序所在目录。  
2. 确认 Excel 文件包含完整数据：
   - 工号、姓名、奖项、是否内定、转盘布局等信息完整无误。

### 2.2 启动程序
1. 双击运行 EXE 文件。  
2. 系统会自动读取 Excel 数据并初始化转盘。

### 2.3 开始抽奖
1. 点击 **开始抽奖** 按钮。  
2. 转盘旋转至中奖格子并弹出中奖信息窗口。  
3. 系统自动将中奖结果记录到 Excel 文件中。

### 2.4 结束抽奖
- 当奖池为空时，系统会提示“抽奖结束”，并停止抽奖操作。  
- 所有中奖记录均已保存，可用于后续统计和核查。

---

## 3. 注意事项
- ⚠️ 请确保 Excel 文件未被其他程序占用，以免写入失败。  
- ⚠️ Excel 文件必须包含完整员工信息和转盘布局，否则可能导致抽奖异常。  
- ⚠️ 中奖记录文件会自动生成或追加，请勿随意删除。  
- ⚠️ 建议在正式抽奖前进行一次测试，确认数据与转盘显示正常。  

---
